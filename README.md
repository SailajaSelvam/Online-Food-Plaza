# Online-Food-Plaza
  Food Plaza System is an online food ordering platform where customers can browse and order food easily, and admins can manage food items and track orders. The system provides secure login, role-based access, and a user-friendly interface.

Features

Customer: Browse food, add to cart, place orders, manage profile

Admin: Add/update/delete food items, track orders, manage profile

Common: Secure login/logout, role-based access, real-time updates

Technology

JSP (JavaServer Pages) – Dynamic web pages

Eclipse IDE – Development environment

MySQL & XAMPP – Database and local server

Database Tables

fooditem, customer, signup, cart – To manage food, users, and orders
